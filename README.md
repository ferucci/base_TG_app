# Telegramm App Clicker!

Инициализация проекта на хостинге:
<firebase init>

Сборка проекта:
<npm run build>

Для изменений выполнить команду:
<firebase deploy>

База данных:
`Supabase`

Подключена библиотека для получения функционала tg:
<script src="https://telegram.org/js/telegram-web-app.js"></script>
